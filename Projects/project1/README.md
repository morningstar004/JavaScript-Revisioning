# JavaScript Background Color Switcher

This project is a simple web application that allows users to change the background color of the page by clicking on color buttons. 

## Project Structure

```
project1
├── src
│   ├── index.html        # Main HTML document
│   ├── styles.css       # Styles for the application
│   └── script.js        # JavaScript for interactivity
├── .gitignore           # Files and directories to ignore by Git
├── package.json         # npm configuration file
└── README.md            # Project documentation
```

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd project1
   ```

3. **Open the `src/index.html` file in your web browser** to view the application.

## Usage

- Click on any of the color buttons (Grey, White, Blue, Yellow) to change the background color of the page.
- The buttons are styled to be visually distinct and easy to interact with.

This project does not have any external dependencies.
